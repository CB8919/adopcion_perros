# Sistema de Adopción de Mascotas

## Descripción

Sistema web orientado a la gestión de adopción de mascotas.

Permite administrar usuarios, solicitudes de adopción y seguimiento del proceso.

## Objetivo

Simular un sistema real donde intervienen distintos roles y validaciones de negocio.

## Funcionalidades

- Registro y login de usuarios
- Publicación de mascotas
- Solicitud de adopción
- Gestión y seguimiento de solicitudes
- Validaciones de datos

## Enfoque funcional

Este sistema fue pensado desde una lógica de negocio:

- Flujo de adopción controlado
- Validación de estados
- Interacción entre usuarios y sistema

## Tecnologías

- Python
- Django
- SQLite
